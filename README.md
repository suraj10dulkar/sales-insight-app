# Sales Insights App


## Quick start


1. Install dependencies:

npm install

2. Start dev server:

npm run dev

3. Open http://localhost:5173 and upload your Excel file.

Notes

Expected columns: Date, Product, Region, Sales, Quantity (fuzzy matching supported).

Add CSV parsing if you need .csv files.
